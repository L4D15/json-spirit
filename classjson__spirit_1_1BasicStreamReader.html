Detailed Description

template<class Istream_type, class ValueType>
class json_spirit::BasicStreamReader< Istream_type, ValueType >

StreamReader lets you read multiple top-level values from a stream. Normally you are only able to parse one top-level object (e.g. a top-level object), but this lets you place them in a stream instead. This class can work with exceptions or by just returning a success code.
